The Bot is Down And I Can't Use It

If the bot is down, I might just testing it. In this case the bot should come online and offline in a short period of time and while it's active it might return something like: Under Testing, Come Back Later.

Another possiblity is that there might be a problem with the hoster, which will result in the bot being completely offline. In this case the bot should be back up and running in the next few hours.

If the bot is down for an extended period of time please let me know via Discord




There is an error displayed when trying to use moderation commands

The error is created when a user attempts to use the bot without the correct permissions or when the bot doesn't have the right permission. This should not be an error for the bot as it asks for administrator permissions but make sure that it has kick/ban permissions. The same goes for members




The bot is in my server but I can't use it

The bot must read your messages to work therefore it must have access to all channels you want to use this in. It also need to be able send messages.
